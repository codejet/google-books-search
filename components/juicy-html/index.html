<juicy-html>

<juicy-html> (precisely: <template is="juicy-html">) is a custom element that lets you load HTML partials into your Web Components. See README.md for more information

Examples

  • <juicy-html> in a <template> with HTML partial from a string
  • <juicy-html> with HTML partial loaded from a file
  • <juicy-html> in a <template> with HTML partial loaded from a file, and attached binding
  • <juicy-html> in a <template> with HTML partial loaded directly to DOM tree (not wrapeed by <juicy-html> node)
  • <juicy-html> in a Polymer element
  • <juicy-html> serving native <template> features and attributes
